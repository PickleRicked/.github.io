How to install a build into Kodi

  • Open up Kodi
  • Click the Settings icon

  • Open File Manager

  • Click Add source

  • Click <None>

    • IMPORTANT!!!

  • Enter the following URL exactly as it's shown:-
    https://picklericked.github.io/r/repo

  • Select the box underneath, enter any name for the source

  • Click OK

  • Press back button on remote or keyboard to return to System page.
    Now click Addons

  • Click Install from Zip File

  • Click settings

  • Go down and click Unknown sources button

  • Click Yes to allow

  • Click Install from respository

  • Select the source under the name you selected

  • Click repository.tomskodi-X.X.zip

  • Wait for Toms Kodi icon to appear, then click Install from Repository

  • Click Toms Kodi

  • Click Program add-ons

  • Click PickleRick Wizard

  • Click Install

  • Wait for PickleRick Wizard Add-on installed message to pop up

  • Click Continue

  • Click: CLICK HERE

  • Click PickleRick Wizard again

  • Click Open

  • Click Builds tab

  • Select the build you want, Click Install;
    or Click Fresh Install to overwrite existing build

  • Click Yes Install

  • Wait for Download & Installation

  • When completed Click Force Close

  • Congratulations, you made it! Any issues... Not my problem!
